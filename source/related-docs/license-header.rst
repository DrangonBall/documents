.. _refLicenseHeader:

==========================
License头模板
==========================

License头必须放在每一个源代码文件的开头。

* Python::

    # SPDX-License-Identifier: Apache-2.0
    #

* Golang::

    /*
     SPDX-License-Identifier: Apache-2.0
    */

* JavaScript::

    /*
    # SPDX-License-Identifier: Apache-2.0
    */
